### Installation:
* Clone the repository to a local directory:
  ```sh
  git clone https://github.com/Stanis96/
  ```
* Set your own variable values in ```.env_template``` and rename to ```.env```
* Application launch:
```sh
  docker-compose -f docker-compose.yaml up --build
  ```
